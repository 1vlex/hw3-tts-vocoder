# REPORT

## Модель
Реализован baseline HiFi-GAN вокодер.

## Архитектура
Generator:
upsampling conv + residual blocks

Discriminators:
MPD + MSD

Loss:
- adversarial
- feature matching
- mel L1

## Данные
Датасет: RUSLAN  
Sample rate: 22050

## Результаты resynthesis
- речь разборчива
- есть роботизированность
- интонация в целом сохраняется

## Анализ MOS данных
- текст различим
- длинные предложения стабильны
- натуральность ограничена

## Full TTS анализ

Pipeline:
text -> external TTS -> mel -> наш вокодер

Наблюдения:
- больше шума чем в resynthesis
- сильнее роботизированность
- частично пропадает интонация
- ошибки ударений
- ухудшается естественность речи

Причина:
ошибки приходят из внешней TTS и усиливаются вокодером

Вывод:
качество полной TTS зависит не только от вокодера,
но и от качества upstream модели.

## Итог
Вокодер:
- работает стабильно
- сохраняет разборчивость
- ограничен по натуральности
